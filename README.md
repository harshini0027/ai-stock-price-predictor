# AI Stock Price Predictor Dashboard

This project is a machine learning-powered dashboard that predicts stock prices using historical market data.

## Features

- Real-time stock data retrieval
- Machine learning prediction using Linear Regression
- Interactive dashboard built with Streamlit
- Stock price visualization

## Technologies Used

Python  
Streamlit  
Scikit-learn  
Pandas  
Matplotlib  
Yahoo Finance API

## How to Run

Install dependencies:

pip install -r requirements.txt

Run the dashboard:

streamlit run stock_dashboard.py
